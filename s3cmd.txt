ставим клиента

можно ствпить через пактеты
 	$ sudo apt-get install s3cmd


но лучше скачать с сайта гитхаб и ставить "руками" более новую версию


	$ wget https://github.com/s3tools/s3cmd/releases/download/v2.4.0/s3cmd-2.4.0.tar.gz
	$ unzip ...
	$ sudo apt-get install python3-setuptools (если это манжаро то это ненадо)
    $ sudo python3 setup.py install (обяательно с sudo !!!! нахрен)




инициализируем клиента

	$ s3cmd --configure

оно спросить у нас api key через котоырй мы будем получать доступ к бакету. 
и прочую хрень. и это все он записывает в файл 

	~/.s3cfg
