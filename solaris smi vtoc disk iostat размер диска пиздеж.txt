вот о чем речь. 
команда format в солярисе
если мы в не смотрим  состав диска и этот диск разбит как SMI\VTOC
то этот мудак показывает неполный обьем диска!
полный обьем можно 4увидеть только в iostat

пример

# iostat -En
c2t2d0           Soft Errors: 0 Hard Errors: 72 Transport Errors: 0 
Vendor: ATA      Product: ST2000NM0033-9ZM Revision: SN06 Serial No: Z1X415WK 
Size: 1000.20GB <1000204886016 bytes>


вот полный обьем диска  в байтах
1000204886016 bytes
это реальный настоящий обьем



а вот какой обьем покзывает format


Current partition table (original):
Total disk cylinders available: 60797 + 2 (reserved cylinders)

Part      Tag    Flag     Cylinders         Size            Blocks
  0       root    wm       1 - 60796      931.44GB    (60796/0/0) 1953375480
  1 unassigned    wm       0                0         (0/0/0)              0
  2     backup    wu       0 - 60796      931.46GB    (60797/0/0) 1953407610
  3 unassigned    wm       0                0         (0/0/0)              0
  4 unassigned    wm       0                0         (0/0/0)              0
  5 unassigned    wm       0                0         (0/0/0)              0
  6 unassigned    wm       0                0         (0/0/0)              0
  7 unassigned    wm       0                0         (0/0/0)              0
  8       boot    wu       0 -     0       15.69MB    (1/0/0)          32130
  9 unassigned    wm       0                0         (0/0/0)              0



значит я сразу раскажу как эту таблицу расшифровать
вверху написано 
	Total disk cylinders available: 60797 + 2 (reserved cylinders)
тоесть диск состоит из 60799 цилиндров


смотрим на саму таблицу в колонку  "Cylinders"
так вот в ней для кадого раздела указано скько цилиндров он  занимает.
логично что раздел занимает только целое число цилиндров
смотрим.


Part      Tag    Flag     Cylinders         Size            Blocks
  8       boot    wu       0 -     0       15.69MB    (1/0/0)          32130


цилиндр 0 заимает раздел 8 он же boot  и справа указано сколько он занимает блоков 32130 (каждый блок
это 512 байт)
таким макаром мы узнали сколько блоков занимает один цидиндр на этом диске


смотрим другой раздел

Part      Tag    Flag     Cylinders         Size            Blocks
  0       root    wm       1 - 60796      931.44GB    (60796/0/0) 1953375480


здест указано что этот разел занимает цилиндры с 1 по 60796 и занимает 1953375480 блоков


есть супер полезный раздел 

Part      Tag    Flag     Cylinders         Size            Blocks
  2     backup    wu       0 - 60796      931.46GB    (60797/0/0) 1953407610


это такой спец раздел . он охватывает собой весь диск целиком . на солярисе при разбиваке диска
через vtoc\smi этот раздел всегда так выглядит что он охывается весь диск целиком.


и еще вспоминаем что еще два спец цилинра входят в состав цилиндра
	Total disk cylinders available: 60797 + 2 (reserved cylinders)
эти цилиндры лежат за цилиндром 60796.
только мы их  в таблице не увидим.

итак получается что согласно формат размер диска это 60797+2 цилиндра
если на цилинщдре 32130 блоков 
то всего блоков


	(60797+2)*32130*512=1 000 177 597 440

и как видно это отличется от того что пишет iostat  1 000 204 886 016

сравниваем

1 000 177 597 440
1 000 204 886 016

итого ориентироваться на рамер диска который собщает программа format в солярисе нихуя ненадо
только iostat
