вопрос -как с линукса подкючиться к ilo remote console которая на базе java ?
ответ - это миссия невыполнима. заставить в линуксе заработать JNLP файл 
на базе java нереально




решение - https://github.com/FNNDSC/hplocons-wine
тоесть надо создать контейнер.
в котором поставить WINE
и через него поставить виндовскую утилиту HPLOCONS
и потом запустить этот контейнер 
и к нему подключиться через vncviwer

и только так это все заработает


как запустить контейнер


  # podman run --rm -p 5901:5900  localhost/fnndsc/hplocons:latest
