есть два типа лейблов или партишен лейблов EFI и SMI(VTOC)
главный признак VTOC это то что будет 8 слайсов.
а еще у него указаны Cylinders а если EFI то будут Sector

вот пример



Part      Tag    Flag     Cylinders         Size            Blocks
  0       root    wm       0 - 14086      136.71GB    (14087/0/0) 286698624
  1 unassigned    wm       0                0         (0/0/0)             0
  2     backup    wm       0 - 14086      136.71GB    (14087/0/0) 286698624
  3 unassigned    wm       0                0         (0/0/0)             0
  4 unassigned    wm       0                0         (0/0/0)             0
  5 unassigned    wm       0                0         (0/0/0)             0
  6 unassigned    wm       0                0         (0/0/0)             0
  7 unassigned    wm       0                0         (0/0/0)             0


также этот пример показывает пример "правильного разбиения" на слайсы

слайс номер 2 всегда это backup и он занимает весь диск целиком.

также важно упомнять что когда делаешь изменние слайса (они зовут его партишеном)
то там есть вопрос про стартовый цилиндр. а потом про размер слайса а не про стоповый цилиндр
поэтому второй цилиндр нужно вводить не номер послоеднего циилиндраа а размер слайса.
например весь диск занимает 14087 цилиндров.
поэтому на второй вопрос мы говоим 14087c и тогда как видно на картинке
слайс начинается с нулевого цилиндра и заканчиыается на цилинре номер 14086
а размер слайса составляет 14087 цилиндров
просто интитивно хочется ввести 14086 и это ошибка.
осоебно если так сделать на слайсе 2 который backup
он тогда сразу начнет ругаться вот так

Warning: Partition with V_BACKUP tag should specify full disk capacity.


тоесть он мне спрашивает про размер слайса а я это вспринимпаю как номер
последнего цилинда

Enter new starting cyl[0]: 
Enter partition size[286698624b, 14087c, 14086e, 139989.56mb, 136.71gb]: 14086c

и он ругается

Warning: Partition with V_BACKUP tag should specify full disk capacity.


потому что нужно было ввести 14087c и он это делает за нас.



следующая ебала это то как на диске гарантрованно стереть EFI лейбл
и гарантрованно создать SMI VTOC лейбл. для этго всего навсего нужно
запустить format с ключе "-e"


    # format -e

и тогда при вызове label
он спросит

[0] SMI Label
[1] EFI Label


и тут мы можем выбрать ! 
а если формат запускать без ключа "-e" то такого вопроса небуде!

к соаженнию как понять на уже готовом диске ккойто  там лейбл малопонятно.
но вроде как SMI VTOC лейбл если есть то на диске будет 8 слайсов ( 0-7)
